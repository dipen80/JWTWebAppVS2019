This jwtTokenOAuth can be customiza in Startup.cs.

In POSTMAN, 
Paste https://localhost:44376/api/User/GetUserDetails to get below data:
{
    "UserID": null,
    "UserName": null,
    "Password": null,
    "Roles": "Application User",
    "Email": "test.test@gmail.com",
    "ContactNumber": "+91-9856984533"
}
To generate token which need to pass to get above data. 
under header tab: Add Key as Authorization and value generated by https://localhost:44376/GenerateToken 
with key/value under X-www-form-urlencoded tab, userid = dipen password = 123 grant_type = password
